# Hashgen
A simple and fast File Hashing utility.

Available Hashes:

                    1. SHA-1 (sha1)
                    
                    2. SHA-256 (sha256)
                    
                    3. SHA-512 (sha512)


# Installation:

```Bash
$ curl https://github.com/shaswata56/Hashgen/releases/download/v1.1/hashgen
$ sudo mv hashgen /bin
```
